### to-json
```red
USAGE:
     TO-JSON data

DESCRIPTION: 
     Convert Red data to a JSON string. 
     TO-JSON is a function! value.

ARGUMENTS:
     data          

REFINEMENTS:
     /pretty      => 
        indent       [string!] {Pretty format the output, using given indentation.}
     /ascii       => Force ASCII output (instead of UTF-8).

```
