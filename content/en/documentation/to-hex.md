### to-hex
```red
USAGE:
     TO-HEX value

DESCRIPTION: 
     Converts numeric value to a hex issue! datatype (with leading # and 0's). 
     TO-HEX is a native! value.

ARGUMENTS:
     value        [integer!] 

REFINEMENTS:
     /size        => Specify number of hex digits in result.
        length       [integer!] 

RETURNS:
     [issue!]
```
