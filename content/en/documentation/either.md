### either
```red
USAGE:
     EITHER cond true-blk false-blk

DESCRIPTION: 
     If conditional expression is truthy, evaluate the first branch; else evaluate the alternative. 
     EITHER is a native! value.

ARGUMENTS:
     cond         [any-type!] 
     true-blk     [block!] 
     false-blk    [block!] 

```
