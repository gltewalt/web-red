### poke
```red
USAGE:
     POKE series index value

DESCRIPTION: 
     Replaces the series value at a given index, and returns the new value. 
     POKE is an action! value.

ARGUMENTS:
     series       [series! port! bitset!] 
     index        [scalar! any-string! any-word! block! logic!] 
     value        [any-type!] 

RETURNS:
     [series! port! bitset!]

```
