### arctangent
```red
USAGE:
     ARCTANGENT tangent

DESCRIPTION: 
     Returns the trigonometric arctangent (in degrees by default in range [-90,90]). 
     ARCTANGENT is a native! value.

ARGUMENTS:
     tangent      [number!] "in range [-inf,+inf]."

REFINEMENTS:
     /radians     => Angle is returned in radians [-pi/2,pi/2].

RETURNS:
     [float!]

```
