### arctangent2
```red
USAGE:
     ARCTANGENT2 y x

DESCRIPTION: 
     Returns the smallest angle between the vectors (1,0) and (x,y) in degrees by default (-180,180]. 
     ARCTANGENT2 is a native! value.

ARGUMENTS:
     y            [number!] 
     x            [number!] 

REFINEMENTS:
     /radians     => Angle is returned in radians (-pi,pi].

RETURNS:
     [float!]

```
