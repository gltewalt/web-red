### arccosine
```red
USAGE:
     ARCCOSINE cosine

DESCRIPTION: 
     Returns the trigonometric arccosine (in degrees by default in range [0,180]). 
     ARCCOSINE is a native! value.

ARGUMENTS:
     cosine       [number!] "in range [-1,1]."

REFINEMENTS:
     /radians     => Angle is returned in radians [0,pi].

RETURNS:
     [float!]

```
