### within?
```red
USAGE:
     WITHIN? point offset size

DESCRIPTION: 
     Return TRUE if the point is within the rectangle bounds. 
     WITHIN? is a function! value.

ARGUMENTS:
     point        [pair!] "XY position."
     offset       [pair!] "Offset of area."
     size         [pair!] "Size of area."

RETURNS:
     [logic!]

```
