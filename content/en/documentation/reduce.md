### reduce
```red
USAGE:
     REDUCE value

DESCRIPTION: 
     Returns a copy of a block, evaluating all expressions. 
     REDUCE is a native! value.

ARGUMENTS:
     value        [any-type!] 

REFINEMENTS:
     /into        => Put results in out block, instead of creating a new block.
        out          [any-block!] "Target block for results, when /into is used."

```
