### arcsine
```red
USAGE:
     ARCSINE sine

DESCRIPTION: 
     Returns the trigonometric arcsine (in degrees by default in range [-90,90]). 
     ARCSINE is a native! value.

ARGUMENTS:
     sine         [number!] "in range [-1,1]."

REFINEMENTS:
     /radians     => Angle is returned in radians [-pi/2,pi/2].

RETURNS:
     [float!]

```
