### lowercase
```red
USAGE:
     LOWERCASE string

DESCRIPTION: 
     Converts string of characters to lowercase. 
     LOWERCASE is a native! value.

ARGUMENTS:
     string       [any-string! char!] "Value to convert (modified when series)."

REFINEMENTS:
     /part        => Limits to a given length or position.
        limit        [number! any-string!] 

RETURNS:
     [any-string! char!]

```
